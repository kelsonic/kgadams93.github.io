The Enumerable Map Method

Iterate, Change, and Save. It's that easy.

Posted March 8, 2016.

Iteration

The Enumerate class in Ruby is incredibly helpful. Chances are you may be familiar with the .each Enumerate method to iterate through, say, an array. You may even want to make changes to the array you are iterating over. If that's the case, then .map is about to become your new best friend.

So what is Enumerate#map?

The .map method takes an enumerable object and a block, and runs the block for each element, outputting each returned value from the block.

It's probably just easiest to give you an example:

arry = [5, 3, 2, 4] puts arry.map { |num| num ** 2 } # => [25, 9, 4, 16]

The .map method will go through each element (or num in this case) in the array and square it.

Don't forget about me!

It's important to note that the new array we just created above will be forgotten unless we use the .map! method, which reassigns the new array back to arry.

And that's about it! If you are curious about more Enumerate methods, definitely check out the Ruby docs here.
